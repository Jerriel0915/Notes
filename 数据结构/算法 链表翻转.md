今天刷到了一道美团面试题，考察的是链表的局部翻转问题：[25. K 个一组翻转链表 - 力扣（LeetCode）](https://leetcode.cn/problems/reverse-nodes-in-k-group/description/?envType=company&envId=meituan&favoriteSlug=meituan-thirty-days)。而局部来源于整体，因此也可以先考虑怎么整体翻转一个链表：

定义一个最基本的链表，包含一个值和指向下一个节点的指针
```cpp
// 单链表节点
struct Node {
    int val;
    Node* next;
    explicit Node(int v) : val(v), next(nullptr) {}
};

// 工具：打印链表
void printList(Node* head) {
    for (Node* p = head; p; p = p->next)
        std::cout << p->val << (p->next ? " -> " : "\n");
}

// 工具：释放链表
void deleteList(Node* head) {
    while (head) {
        Node* tmp = head;
        head = head->next;
        delete tmp;
    }
}
```

## 头插法/尾插法

首先最朴素的想法，可以新建一个头指针，然后遍历链表，每走一步就把当前节点拆下来，使其next指针指向我们构建的新的链表。这样当我们走完一遍后，新的翻转后的链表自然就形成了。

于是就有了**头插法：**

```cpp
// 1. 头插法
Node *reverseBuild(Node *head){
    if (!head || !head->next)
        return head; 			 // 空或只有一个节点

    Node *newHead = nullptr;
    while (head) {
        Node *next = head->next; // 保存后续
        head->next = newHead;    // 插到新头
        newHead = head;          // 更新头指针
        head = next;             // 后移遍历
    }
    return newHead;
}
```

- •
    空间复杂度：O(1)

## 三指针

如果**只能在原链表上修改**，我们可以依次遍历整个链表，每走一步就把当前的节点指向之前的节点，这样当我们走到链表末尾时，我们所处的节点就是新的链表头节点。

这就是**迭代翻转（三指针）法：**
```cpp
// 2. 迭代翻转（三指针）
Node* reverseIterative(Node* head) {
	if (!head || !head->next) 
		return head; 			// 空或只有一个节点

    Node *prev = nullptr;
    Node *curr = head;
    Node *next = nullptr;

    while (curr) {
        next = curr->next;      // 先保存下一个
        curr->next = prev;      // 翻转当前指针
        prev = curr;            // 前指针后移
        curr = next;            // 当前指针后移
    }
    return prev;            // 此时，curr已为空，prev为末尾元素，即新的头指针
}
```

- •
    空间复杂度：O(1)

## 递归法

我们可以将翻转整个链表这个大任务拆分成嵌套的小任务。

想想当只有两个节点的时候我们怎么做的翻转——只将后续节点看做新的头节点，指向原先的头节点，再断开原先的指向就行了。那么我们不妨也将链表看做只有两个节点：当前节点和所有的后续节点。在当前这一步中，我只管将后续节点的next指向自己，此时后续节点就成了头节点。然后还需要返回新的尾节点，作为上层嵌套的头节点，这样链表才能连上。

那么只要保证在翻转之前，后续节点已经完成了翻转就行了，这就是**递归：**
```cpp
// 3. 递归法
Node *reverseRecursive(Node *head) {
    if (!head || !head->next)
        return head;                              	// 空或只有一个节点，递归出口

    Node *newHead = reverseRecursive(head->next); 	// 进入递归
	// 此时后续所有节点已经翻转完成了
    head->next->next = head;        				// 将下一个节点指向自己
    head->next = nullptr;          					// 断开原先的指向

    return newHead;         // 返回新的头节点
}
```

- •
    空间复杂度：O(N) 节点数量过大可能会爆栈

## 压栈出栈

翻转的思想很适合使用栈这个数据结构。遍历一次原链表，依次压栈，此时弹栈顺序就是我们要的逆序，将next重新串联就好了。
```cpp
// 栈辅助法
Node *reverseWithStack(Node *head) {
    if (!head || !head->next)
        return head;            // 空或只有一个节点

    std::stack<Node*> st;
    // 依次压栈
    while (head) {
        st.push(head);
        head = head->next;
    }
    // 新的头
    Node *newHead = st.top();
    st.pop();
    Node *tail = newHead;
    // 依次弹出，连接
    while (!st.empty()) {
        tail->next = st.top();
        st.pop();
        tail = tail->next;
    }
    // 记得断后
    tail->next = nullptr;

    return newHead;
}
```

空间复杂度：O(N) 需要构建一个辅助栈

那么这就是如何对一个链表进行整体翻转，而考虑题目要求的局部翻转，则还需要传入左右区间，但算法思想与上述类似：
```cpp
Node* myReverse(Node* head, Node* left, Node* right) {
    if (!head || left == right) return head;

    Node dummy(0);
    dummy.next = head;
    Node* preLeft = &dummy;        // 左边界的“前一个”节点

    // 1. 让 preLeft 指向 left 的前驱
    while (preLeft->next != left) preLeft = preLeft->next;

    // 2. 就地反转 [left, right] 这一段
    Node* prev = nullptr;          // 反转时的前驱
    Node* curr = left;             // 当前正在处理的节点
    Node* postRight = right->next; // 保存右边界之后的第一节点

    while (curr != postRight) {    // 一直反转到 right 之后停下
        Node* next = curr->next;
        curr->next = prev;
        prev = curr;
        curr = next;
    }

    // 3. 把反转后的子链重新接回原链
    preLeft->next = right;   // right 成了新头
    left->next = postRight;  // left 成了新尾，接后面

    return dummy.next;
}
```

此题题解：
```cpp
class Solution {
public:
    ListNode* reverseKGroup(ListNode* head, int k) {
        if (k == 1) {
            return head;
        }
        ListNode dummy(0);          // 统一头节点可能变更的情况
        dummy.next = head;
        ListNode *prevEnd = &dummy; // 上一段翻转后的尾节点，即当前段的前驱

        while (1) {
            // 量出这一段的长度，不足 k 就结束
            ListNode *tail = prevEnd;       // tail记录旧尾
            for (int i = 0; i < k; ++i) {
                tail = tail->next;
                if (!tail) return dummy.next;   // 剩余不足 k
            }
            // 记录下一段的开头
            ListNode *nextHead = tail->next;
            // 翻转[prevEnd->next, tail]这一段
            ListNode *pre = prevEnd;
            ListNode *curr = prevEnd->next;

            while (curr != nextHead) {
                ListNode *next = curr->next;
                curr->next = pre;
                pre = curr;
                curr = next;
            }
            // 接回主链
            ListNode *oldHead = prevEnd->next;      // 旧头
            prevEnd->next = tail;
            oldHead->next = nextHead;

            prevEnd = oldHead;
        }
    }
};
```
